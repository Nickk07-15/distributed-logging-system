# dls-log-monitoring

This directory contains the Query API service for the distributed logging system.

## Purpose
The query API exposes endpoints for searching and retrieving logs from Elasticsearch. It provides a user-friendly interface for querying logs and visualizing results.

## Responsibilities
- Provide REST API endpoints for log search and retrieval
- Query Elasticsearch for logs based on user input
- Format and return results to clients

## Technologies
- Python (FastAPI or Flask)
- Python client: `elasticsearch`
