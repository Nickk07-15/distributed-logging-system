# dls-elasticsearch

This directory contains the Elasticsearch service for the distributed logging system.

## Purpose
Elasticsearch is used as the storage and search backend for logs. It indexes logs received from the Flink job and allows fast querying and filtering of log data.

## Responsibilities
- Store logs in a searchable format
- Provide APIs for searching and aggregating logs
- Support efficient retrieval for the query API

## Technologies
- Elasticsearch
- Python client: `elasticsearch`
